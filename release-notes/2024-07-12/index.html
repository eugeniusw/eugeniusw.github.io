What’s changed in 2024-07-12 Updates?

  • Changed overall design theme and added dark mode. You can use toggle button at the top right of navbar to switch from dark to light mode.
  • Changed the navigation that reflect files in the source directory.
  • Now search will be much faster to retrieve, since it only fetch the required block data. Thanks to Pagefind!
  • Optimize image based on device screen size and device pixel ratio on build.
  • Optimize html, css, and js assets by minifying and shortened css id, classname and variables. This is how big tech did it. Try to inspect https://google.com or https://facebook.com webpage and you will see.
  • Added table of contents in the article. You can jump to the content by clicking the item.
  • Added 3 types of media screen: for 0px-667px, 668px-922px, and 923px onwards. Mainly for left and right sidebar for easier navigation on large devices. Fun fact! those numbers are the golden ratio of 1080px.
  • Added image caption below the image from the alt text. Only applies for the alt text in original html and not modified by javascript. Use view-source:url to debug.
  • Create release notes. Now every updates of this site will be recorded here.
