Python Basics

Python is a versatile programming language. Let’s start with the basics.

  • Variables and Data Types: Variables store data values, and Python supports various data types like integers, floats, strings, and booleans.

    x = 10            # Integer
y = 3.14          # Float
name = "Alice"    # String
is_active = True  # Boolean

  • Basic Operations: Perform arithmetic and logical operations using operators.

    z = x + y  # Addition
print(z)   # Output: 13.14

  • Input and Output: Get input from users and print output to the console.

    name = input("Enter your name: ")
print(f"Hello, {name}!")

Control Flow

Control flow statements determine the path of execution based on conditions.

  • If-Else Statement: Use conditional statements to make decisions.

    age = 18
if age >= 18:
    print("Adult")
else:
    print("Not an Adult")

  • For Loop: Iterate through sequences like lists, tuples, strings, etc.

    for i in range(5):
    print(i)  # Output: 0, 1, 2, 3, 4

  • While Loop: Loop as long as a condition is true.

    count = 0
while count < 5:
    print(count)
    count += 1

Functions

Functions are reusable blocks of code that perform a specific task.

  • Defining and Calling Functions: Create functions to encapsulate code.

    def greet(name):
    return f"Hello, {name}!"
print(greet("Alice"))  # Output: Hello, Alice!

  • Lambda Functions: Anonymous functions defined with the lambda keyword.

    square = lambda x: x ** 2
print(square(5))  # Output: 25

Data Structures

Python provides various data structures to store collections of data.

  • List: Ordered and mutable collection of items.

    fruits = ["apple", "banana", "cherry"]
fruits.append("orange")
print(fruits)  # Output: ['apple', 'banana', 'cherry', 'orange']

  • Dictionary: Unordered collection of key-value pairs.

    person = {"name": "John", "age": 30}
print(person["name"])  # Output: John

  • Set: Unordered collection of unique items.

    numbers = {1, 2, 3, 2}
print(numbers)  # Output: {1, 2, 3}

  • Tuple: Ordered and immutable collection of items.

    point = (10, 20)
print(point[0])  # Output: 10

  • Collections Module: Provides specialized container data types.

    from collections import namedtuple, deque, Counter

Point = namedtuple('Point', ['x', 'y'])
p = Point(1, 2)
print(p.x, p.y)  # Output: 1 2

d = deque([1, 2, 3])
d.appendleft(0)
print(d)  # Output: deque([0, 1, 2, 3])

counter = Counter(['a', 'b', 'a'])
print(counter)  # Output: Counter({'a': 2, 'b': 1})

  • Iterators and Generators: Objects that you can loop through, typically using the yield keyword for generators.

    def my_generator():
    yield 1
    yield 2
    yield 3

for value in my_generator():
    print(value)  # Output: 1, 2, 3

Strings and Regular Expressions

Strings are sequences of characters, and regular expressions (regex) help find patterns in text.

  • String Methods: Perform operations like conversion, search, and formatting on strings.

    s = "hello"
print(s.upper())  # Output: hello

  • String Formatting: Format strings using f-strings, format method, or old-style formatting.

    age = 25
print(f"I am {age} years old.")  # Output: I am 25 years old.

  • Regular Expressions: Match patterns in strings.

    import re

pattern = r"\d+"
text = "There are 3 apples"
match = re.search(pattern, text)
print(match.group())  # Output: 3

File Handling

File handling allows reading from and writing to files.

  • Reading Files: Open a file, read its contents, and close it automatically using with.

    with open("file.txt", "r") as file:
    content = file.read()
    print(content)

  • Writing to Files: Write data to a file, overwriting it if it exists.

    with open("output.txt", "w") as file:
    file.write("Hello, World!")

  • Exception Handling: Handle errors that occur during file operations.

    try:
    with open("non_existent.txt", "r") as file:
        content = file.read()
except FileNotFoundError:
    print("File not found!")

Object-Oriented Programming (OOP)

OOP allows structuring programs using objects that represent real-world entities.

  • Defining Classes: Define a class with attributes and methods.

    class Dog:
    def __init__(self, name):
        self.name = name

    def bark(self):
        return f"{self.name} is barking!"

dog = Dog("Buddy")
print(dog.bark())  # Output: Buddy is barking!

  • Inheritance: Create a new class based on an existing class.

    class Labrador(Dog):
    def bark(self):
        return f"{self.name} is barking! I am a Labrador."

lab = Labrador("Max")
print(lab.bark())  # Output: Max is barking! I am a Labrador.

Working with Data

Python offers powerful libraries for data manipulation and analysis.

  • NumPy Arrays: Perform operations on arrays using the NumPy library.

    import numpy as np

arr = np.array([1, 2, 3, 4])
print(arr.mean())  # Output: 2.5

  • Pandas DataFrame: Work with tabular data using the Pandas library.

    import pandas as pd

data = {"Name": ["John", "Alice"], "Age": [28, 24]}
df = pd.DataFrame(data)
print(df.head())  # Display the first few rows of the DataFrame

  • Plotting with Matplotlib: Visualize data with the Matplotlib library.

    import matplotlib.pyplot as plt

plt.plot([1, 2, 3], [4, 5, 6])
plt.show()

Web Development

Python can be used to develop web applications and APIs.

  • Simple Web App with Flask: Create a basic web server using Flask.

    from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Hello, Flask!"

if __name__ == "__main__":
    app.run(debug=True)

  • Handling HTTP Requests: Responding to other request methods (POST).

    @app.route("/submit", methods=["POST"])
def submit():
    data = request.form["data"]
    return f"Data received: {data}"

Testing

Testing ensures that your code works as expected.

  • Testing with unittest: Create unit tests with the unittest library.

    import unittest

def add(a, b):
    return a + b

class TestMath(unittest.TestCase):
    def test_add(self):
        self.assertEqual(add(1, 2), 3)

if __name__ == "__main__":
    unittest.main()

  • Testing with pytest: A simple testing framework.

    def test_add():
    assert add(1, 2) == 3
