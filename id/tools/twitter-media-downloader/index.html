Pengunduh Media Twitter

Unduh Media (Video, GIF, Gambar) dari Twitter (X)

Memuat media...

Pengunduh Media Twitter - Cara Menggunakan

Selamat datang di Pengunduh Media Twitter! Alat ini memungkinkan Anda mengunduh video, GIF, dan gambar dari Twitter (X) dengan mudah.

Masukkan URL Media (tautan Tweet)

Mulailah dengan menyalin URL tweet yang ingin Anda unduh. Tempel URL ini ke kolom input di bagian atas halaman.

Anda dapat menemukan dan menyalin tautan dari bilah URL di bagian atas halaman.

Contoh tautan:

https://x.com/elonmusk/status/1820685675899072930

Atau, Anda dapat menyalin tautan dari antarmuka pengguna Twitter (X).

Mulai Proses Pengunduhan

Klik tombol “Unduh” untuk memulai. Proses memuat yang berputar akan muncul, yang menunjukkan bahwa alat ini sedang mengambil media. Harap tunggu hingga proses ini selesai.

Pilih Opsi Pengunduhan Anda

Setelah media berhasil diambil, daftar opsi pengunduhan yang tersedia akan muncul. Anda dapat memilih media berdasarkan ukuran dan kualitas. Klik opsi pengunduhan yang paling sesuai dengan kebutuhan Anda.

Penanganan Kesalahan

Jika terjadi masalah saat mengambil media, pesan kesalahan akan ditampilkan di bawah tombol unduh berwarna merah. Anda dapat mencoba memasukkan kembali linknya atau memeriksa ulang untuk dikoreksi.

Simpan Media Anda

Setelah Anda memilih dan mengeklik opsi unduhan yang Anda inginkan, media akan terbuka di tab baru, tempat Anda dapat mengunduhnya ke perangkat Anda.

Catatan Pembaruan

  • Menekan Enter untuk mendownload sekarang berfungsi.

[2024-09-03]

  • Memperbaiki input link yang tertutupi keyboard di ponsel.
  • Menambahkan dukungan untuk Twitter Card (media tersemat dari website lain).
