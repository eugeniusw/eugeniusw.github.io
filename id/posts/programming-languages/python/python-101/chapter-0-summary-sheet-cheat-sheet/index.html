Dasar-dasar Python

Python adalah bahasa pemrograman serbaguna. Mari kita mulai dengan dasar-dasarnya.

  • Variabel dan Tipe Data: Variabel menyimpan nilai data, dan Python mendukung berbagai tipe data seperti integer, float, string, dan boolean.

    x = 10            # Integer 
y = 3.14          # Float 
name = "Alice"    # String 
is_active = True  # Boolean

  • Operasi Dasar: Lakukan operasi aritmatika dan logika menggunakan operator.

    z = x + y  # Penjumlahan 
print(z)   # Output: 13.14

  • Input dan Output: Dapatkan input dari pengguna dan cetak output ke konsol.

    name = input("Masukkan nama Anda: ") 
print(f"Halo, {name}!")

Alur Kontrol

Pernyataan alur kontrol menentukan jalur eksekusi berdasarkan kondisi.

  • Pernyataan If-Else: Gunakan pernyataan kondisional untuk membuat keputusan.

    age = 18 
if age >= 18:     
    print("Dewasa") 
else:     
    print("Belum Dewasa")

  • For Loop: Iterasi melalui urutan seperti list, tuple, string, dll.

    for i in range(5):     
    print(i)  # Output: 0, 1, 2, 3, 4

  • While Loop: Lakukan perulangan selama kondisi bernilai benar.

    count = 0 
while count < 5:     
    print(count)     
    count += 1

Fungsi

Fungsi adalah blok kode yang dapat digunakan kembali untuk melakukan tugas tertentu.

  • Mendefinisikan dan Memanggil Fungsi: Buat fungsi untuk mengenkapsulasi kode.

    def greet(name):     
    return f"Halo, {name}!"  

print(greet("Alice"))  # Output: Halo, Alice!

  • Fungsi Lambda: Fungsi anonim yang didefinisikan dengan kata kunci lambda.

    square = lambda x: x ** 2 
print(square(5))  # Output: 25

Struktur Data

Python menyediakan berbagai struktur data untuk menyimpan koleksi data.

  • List: Koleksi item yang terurut dan dapat diubah.

    fruits = ["apel", "pisang", "ceri"] 
fruits.append("jeruk") 
print(fruits)  # Output: ['apel', 'pisang', 'ceri', 'jeruk']

  • Dictionary: Koleksi pasangan kunci-nilai yang tidak terurut.

    person = {"nama": "John", "usia": 30} 
print(person["nama"])  # Output: John

  • Set: Koleksi item unik yang tidak terurut.

    numbers = {1, 2, 3, 2} 
print(numbers)  # Output: {1, 2, 3}

  • Tuple: Koleksi item yang terurut dan tidak dapat diubah.

    point = (10, 20) 
print(point[0])  # Output: 10

  • Modul Collections: Menyediakan tipe data wadah khusus.

    from collections import namedtuple, deque, Counter  

Point = namedtuple('Point', ['x', 'y']) 
p = Point(1, 2) 
print(p.x, p.y)  # Output: 1 2  

d = deque([1, 2, 3]) 
d.appendleft(0) 
print(d)  # Output: deque([0, 1, 2, 3])  

counter = Counter(['a', 'b', 'a']) 
print(counter)  # Output: Counter({'a': 2, 'b': 1})

  • Iterator dan Generator: Objek yang dapat Anda loop melalui, biasanya menggunakan kata kunci yield untuk generator.

    def my_generator():     
    yield 1     
    yield 2     
    yield 3  

for value in my_generator():     
    print(value)  # Output: 1, 2, 3

String dan Regular Expression

String adalah urutan karakter, dan regular expression (regex) membantu mencari pola dalam teks.

  • Metode String: Lakukan operasi seperti konversi, pencarian, dan format pada string.

    s = "halo" 
print(s.upper())  # Output: halo

  • Pemformatan String: Format string menggunakan f-string, metode format, atau pemformatan gaya lama.

    age = 25 
print(f"Saya berusia {age} tahun.")  # Output: Saya berusia 25 tahun.

  • Regular Expression: Cocokkan pola dalam string.

    import re 

pattern = r"\d+" 
text = "Ada 3 apel" 
match = re.search(pattern, text) 
print(match.group())  # Output: 3

Penanganan File

Penanganan file memungkinkan membaca dari dan menulis ke file.

  • Membaca File: Buka file, baca isinya, dan tutup secara otomatis menggunakan with.

    with open("file.txt", "r") as file:     
    content = file.read()     
    print(content)

  • Menulis ke File: Tulis data ke file, menimpa jika sudah ada.

    with open("output.txt", "w") as file:     
    file.write("Halo, Dunia!")

  • Penanganan Eksepsi: Tangani kesalahan yang terjadi selama operasi file.

    try:     
    with open("tidak_ada.txt", "r") as file:         
        content = file.read() 
except FileNotFoundError:     
    print("File tidak ditemukan!")

Pemrograman Berorientasi Objek (OOP)

OOP memungkinkan menyusun program menggunakan objek yang mewakili entitas dunia nyata.

  • Mendefinisikan Kelas: Definisikan kelas dengan atribut dan metode.

    class Dog:     
    def __init__(self, name):         
        self.name = name      

    def bark(self):         
        return f"{self.name} menggonggong!"  

dog = Dog("Buddy") 
print(dog.bark())  # Output: Buddy menggonggong!

  • Inheritance: Buat kelas baru berdasarkan kelas yang sudah ada.

    class Labrador(Dog):     
    def bark(self):         
        return f"{self.name} menggonggong! Saya adalah Labrador."  

lab = Labrador("Max") 
print(lab.bark())  # Output: Max menggonggong! Saya adalah Labrador.

Bekerja dengan Data

Python menawarkan perpustakaan yang kuat untuk manipulasi dan analisis data.

  • Array Numpy: Lakukan operasi pada array menggunakan perpustakaan NumPy.

    import numpy as np 

arr = np.array([1, 2, 3, 4]) 
print(arr.mean())  # Output: 2.5

  • Pandas DataFrame: Bekerja dengan data tabular menggunakan perpustakaan Pandas.

    import pandas as pd 

data = {"Nama": ["John", "Alice"], "Usia": [28, 24]} 
df = pd.DataFrame(data) 
print(df.head())  # Menampilkan beberapa baris pertama dari DataFrame

  • Plotting dengan Matplotlib: Visualisasikan data dengan perpustakaan Matplotlib.

    import matplotlib.pyplot as plt 

plt.plot([1, 2, 3], [4, 5, 6]) 
plt.show()

Pengembangan Web

Python dapat digunakan untuk mengembangkan aplikasi web dan API.

  • Aplikasi Web Sederhana dengan Flask: Buat server web dasar menggunakan Flask.

    from flask import Flask  

app = Flask(__name__)  

@app.route("/") 
def home():     
    return "Halo, Flask!"  

if __name__ == "__main__":     
    app.run(debug=True)

  • Menangani Permintaan HTTP: Menanggapi metode permintaan lainnya (POST).

    @app.route("/submit", methods=["POST"]) 
def submit():     
    data = request.form["data"]     
    return f"Data diterima: {data}"

Testing

Testing memastikan bahwa kode Anda bekerja seperti yang diharapkan.

  • Pengujian dengan unittest: Buat pengujian unit dengan perpustakaan unittest.

    import unittest  

def add(a, b):     
    return a + b  

class TestMath(unittest.TestCase):     
    def test_add(self):         
        self.assertEqual(add(1, 2), 3)  

if __name__ == "__main__":     
    unittest.main()

  • Testing dengan pytest: Framework testing yang sederhana.

    def test_add():     
    assert add(1, 2) == 3
