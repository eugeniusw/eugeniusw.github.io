Membuat Tata Letak Dua Kolom dengan Guliran Tersinkronisasi

web-dev

  • body, html diatur tanpa margin atau padding dan mengambil seluruh tinggi tampilan.

  • Gambar di dalam .horizontal-scroll ditata untuk memiliki tinggi yang konsisten dan jarak antar gambar.

JavaScript untuk Pengguliran Tersinkronisasi

Untuk menyinkronkan pengguliran konten utama dan gambar, kita menggunakan JavaScript untuk menghubungkan pengguliran vertikal konten utama dengan pengguliran horizontal gambar secara proporsional.

Javascript

document.addEventListener('DOMContentLoaded', function() {
    const content = document.getElementById('content');
    const horizontalScroll = document.getElementById('pictures');

    content.addEventListener('scroll', function() {
        const contentHeight = content.scrollHeight - content.clientHeight;
        const scrollRatio = content.scrollTop / contentHeight;
        
        const horizontalScrollWidth = horizontalScroll.scrollWidth - horizontalScroll.clientWidth;
        horizontalScroll.scrollLeft = scrollRatio * horizontalScrollWidth;
    });
});
  • Kita menambahkan event listener ke div .content untuk mendeteksi acara pengguliran.
  • contentHeight dihitung sebagai total tinggi yang dapat digulir dari konten utama dikurangi tinggi yang terlihat.
  • scrollRatio adalah posisi gulir vertikal saat ini dibagi dengan total tinggi yang dapat digulir.
  • horizontalScrollWidth dihitung sebagai total lebar yang dapat digulir dari area gulir horizontal dikurangi lebar yang terlihat.
  • horizontalScroll.scrollLeft diatur ke scrollRatio dikalikan dengan horizontalScrollWidth, memastikan bahwa posisi gulir horizontal sebanding dengan posisi gulir vertikal.

Kesimpulan

Dengan pengaturan ini, kita telah berhasil membuat tata letak dua kolom yang responsif di mana konten utama bergulir secara vertikal dan gambar yang berdekatan bergulir secara horizontal secara paralel. Kode JavaScript memastikan bahwa kedua kolom mencapai titik akhir mereka secara bersamaan, memberikan pengalaman pengguliran yang mulus dan tersinkronisasi. Anda dapat menyesuaikan properti CSS agar sesuai dengan kebutuhan desain Anda dan menambahkan lebih banyak konten dan gambar untuk melihat efeknya.
