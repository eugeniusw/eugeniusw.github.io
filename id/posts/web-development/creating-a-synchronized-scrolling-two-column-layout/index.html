Eugenius Wahyudiarto Agu 6, 2024
web-dev

JavaScript untuk Pengguliran Tersinkronisasi

Untuk menyinkronkan pengguliran konten utama dan gambar, kita menggunakan JavaScript untuk menghubungkan pengguliran vertikal konten utama dengan pengguliran horizontal gambar secara proporsional.

Javascript

document.addEventListener('DOMContentLoaded', function() {
    const content = document.getElementById('content');
    const horizontalScroll = document.getElementById('pictures');

    content.addEventListener('scroll', function() {
        const contentHeight = content.scrollHeight - content.clientHeight;
        const scrollRatio = content.scrollTop / contentHeight;
        
        const horizontalScrollWidth = horizontalScroll.scrollWidth - horizontalScroll.clientWidth;
        horizontalScroll.scrollLeft = scrollRatio * horizontalScrollWidth;
    });
});
  • Kita menambahkan event listener ke div .content untuk mendeteksi event pengguliran.
  • contentHeight dihitung sebagai total tinggi konten utama yang dapat digulir dikurangi tinggi yang terlihat.
  • scrollRatio ditentukan dengan membagi posisi pengguliran vertikal saat ini dengan total tinggi yang dapat digulir.
  • horizontalScrollWidth adalah total lebar area pengguliran horizontal yang dapat digulir dikurangi lebar yang terlihat.
  • Kita menetapkan horizontalScroll.scrollLeft ke scrollRatio yang dikalikan dengan horizontalScrollWidth, yang memastikan posisi pengguliran horizontal sesuai dengan pengguliran vertikal secara proporsional.

Kesimpulan

Dengan pengaturan ini, kita berhasil membuat tata letak dua kolom yang responsif, di mana konten utama digulir secara vertikal sementara gambar yang berdekatan digulir secara horizontal secara sinkron. Kode JavaScript memastikan bahwa kedua kolom mencapai titik akhirnya secara bersamaan, sehingga memberikan pengalaman pengguliran yang lancar dan tersinkronisasi. Anda dapat menyesuaikan properti CSS agar sesuai dengan kebutuhan desain Anda dan menambahkan lebih banyak konten dan gambar untuk melihat efeknya secara langsung.
